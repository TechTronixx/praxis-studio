01 / problem

A logistics platform stuck in 2003.

Desktop-only, no API, dispatchers printing screenshots to share status. Forty trucks and a clipboard workflow that could not scale past fifty.

02 / approach

Embed with the dispatchers for two weeks.

Our engineers rode along. Learned the real workflow, not the documented one. The dispatchers had built shortcuts the original software never accounted for.

03 / build

A web app drivers can use from a phone.

Real-time dispatch dashboard, driver mobile app, automated status updates. Same data, same workflows, zero retraining. Replaced the old system in one cutover.

04 / result

50 to 200 trucks in six months.

Dispatch time down 60%. Driver status visible in real time. The platform now handles 200 trucks with the same three dispatchers who were there at 50.

Ledger OS

Fintech · Product & Engineering

A banking platform for a new generation of financial institutions. We designed the entire interface, built the component library, and shipped the first version in six months.

The core challenge was making complex financial data readable at a glance. We stripped every screen down to its essential elements , what the Greeks would call getting to the πρᾶξις of the thing.

Field Atlas

Mapping · Design System

A design system and mapping toolkit for an enterprise GIS platform. Thousands of components, dozens of themes, and a documentation engine that writes itself from the code.

The system serves 40+ product teams across three continents. Consistency at that scale is not about rules , it is about making the right thing the easy thing.

Case study

Cadence

Health · iOS & API

An iOS app for tracking chronic pain patterns, built with a privacy-first architecture that keeps all data on-device. The companion API handles only anonymized, opt-in research contributions.

Healthcare apps tend to over-explain. We took the opposite approach: one screen, one question, no friction. The complexity lives in the algorithms, not the interface.

Northwind

Logistics · Web Platform

A real-time logistics dashboard and dispatch platform for a mid-continent freight operator. Replaced a 20-year-old desktop system with a web app that drivers can use from a phone.

The hardest part was not the real-time data pipeline , it was making sure a dispatcher on a 12-hour shift could use it without thinking. Every pixel had to earn its place.
